# Pharma Manufacturing Support Agent

AI-powered support agent designed for pharma manufacturing workflows. This system simulates how operators and analysts can retrieve SOP guidance and receive context-aware recommendations.

## Features
- SOP-based guidance simulation
- API-first architecture using FastAPI
- Designed for SAP and MES workflow support
- Extendable for GxP-compliant environments

## Run Locally

```bash
pip install -r requirements.txt
uvicorn main:app --reload
